# Meteor React Scaffold

### What's in the box?
  - Autoprefixer
  - SCSS
  - Roles
  - Accounts (password)
  - FlowRouter
  - React
  - Griddr (my SCSS grid library)
  - eslint config
  - editor config

### How do I start?
  - Global search for `__METEOR-REACT-SCAFFOLD__` and replace with your project's name
  - `meteor npm install`
  - `meteor`
